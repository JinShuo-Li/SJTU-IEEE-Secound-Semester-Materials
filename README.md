## Overview

This repository collects lecture notes, assignments, code, datasets, and supporting materials for multiple courses. It includes both English and Chinese file/folder names to preserve the original sources and course materials.

## Naming Conventions

- Top-level folders follow the pattern: `<prefix>-<course code>-<course name>/`.
- Common file and folder conventions:
  - `note.md` or `note/`: course notes in Markdown
  - `assignment*/`, `Homework/`: assignments, tests, and judger scripts
  - `Prg/`, `code_*/`: course-related code
  - `images/`, `figures/`: illustrations and figures
