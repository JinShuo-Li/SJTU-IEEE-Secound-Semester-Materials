基本概念

常微分方程的基本定义

现在我们给出对于常微分方程的最一般的定义:

定义(常微分方程): 对于联系一个函数自变量和这个函数以及它的nnn阶导数的方程, 形如:

F(x,y,y′,y′′,⋯ ,y(n))=0F(x, y, y', y'', \cdots, y^{(n)}) = 0F(x,y,y′,y′′,⋯,y(n))=0

我们称之为一个nnn阶常微分方程. 其中xxx是自变量, yyy是未知函数. nnn是一个正整数, 代表了这个方程的阶数. FFF是一个已知的函数.

若常微分方程中, 未知函数yyy和它的各阶导数都以一次幂的形式出现, 则称这个方程是线性的. 否则, 就是非线性的.

例如: y′+p(x)y=q(x)y' + p(x)y = q(x)y′+p(x)y=q(x)是一个一阶线性常微分方程.

定义(常微分方程的解): 如果一个函数y=ϕ(x)y = \phi(x)y=ϕ(x)在区间JJJ上满足常微分方程, 那么我们称这个函数是这个常微分方程在区间JJJ上的一个解.

定义(常微分方程的通解): 如果一个函数y=ϕ(x,C1,C2,⋯ ,Cn)y = \phi(x, C_1, C_2, \cdots, C_n)y=ϕ(x,C1​,C2​,⋯,Cn​)包含了nnn个任意常数C1,C2,⋯ ,CnC_1, C_2, \cdots, C_nC1​,C2​,⋯,Cn​, 并且当我们将这个函数代入常微分方程时, 方程恒成立. 那么我们称这个函数是这个常微分方程的一个通解.

定义(常微分方程的特解): 如果一个函数y=ϕ(x)y = \phi(x)y=ϕ(x)是常微分方程的一个解, 并且这个函数中不包含任何任意常数, 那么我们称这个函数是这个常微分方程的一个特解.

为了从通解得到符合具体图景的特解, 我们需要引入初值的概念.

定义(常微分方程的初值): 对于一个nnn阶常微分方程, 如果我们给定了一个点x0x_0x0​以及未知函数yyy在这个点的值y(x0)=y0y(x_0) = y_0y(x0​)=y0​, 以及未知函数yyy在这个点的各阶导数的值y′(x0)=y1,y′′(x0)=y2,⋯ ,y(n−1)(x0)=yn−1y'(x_0) = y_1, y''(x_0) = y_2, \cdots, y^{(n-1)}(x_0) = y_{n-1}y′(x0​)=y1​,y′′(x0​)=y2​,⋯,y(n−1)(x0​)=yn−1​, 那么我们称这个集合(x0,y0,y1,⋯ ,yn−1)(x_0, y_0, y_1, \cdots, y_{n-1})(x0​,y0​,y1​,⋯,yn−1​)是这个常微分方程的一个初值.

微分方程及其解的几何解释

为了简化, 我们先考虑一阶微分方程:

dydx=f(x,y)\frac{dy}{dx} = f(x, y)dxdy​=f(x,y)

假设y=ϕ(x)y = \phi(x)y=ϕ(x)是这个微分方程的一个解, 那么在坐标平面上, 这个解就表示为一条曲线, 我们称这条曲线为微分方程的积分曲线.

代入方程的解, 我们可以得到:

dϕdx=ϕ′(x)=f(x,ϕ(x))\frac{d\phi}{dx} = \phi '(x) = f(x, \phi(x))dxdϕ​=ϕ′(x)=f(x,ϕ(x))

因此我们可以知道积分曲线在P0(x0,y0)P_0(x_0, y_0)P0​(x0​,y0​)点的切线斜率是f(x0,y0)f(x_0, y_0)f(x0​,y0​). 切线方程可以写成:

y−y0=f(x0,y0)(x−x0)y - y_0 = f(x_0, y_0)(x - x_0)y−y0​=f(x0​,y0​)(x−x0​)

这样, 在区域DDD内的每一个点P(x,y)P(x, y)P(x,y), 都有一个对应的切线斜率f(x,y)f(x, y)f(x,y). 这就形成了一个方向场. 方向场可以帮助我们理解微分方程的解的行为. 例如, 如果在某个区域内, 方向场的斜率都很大, 那么这个区域内的积分曲线可能会非常陡峭. 反之, 如果在某个区域内, 方向场的斜率都很小, 那么这个区域内的积分曲线可能会比较平缓. 通过分析方向场, 我们可以获得关于微分方程解的定性信息, 例如解的稳定性, 周期性等性质.

我们可以通过方向场近似的画出微分方程的积分曲线. 例如, 如果我们知道在某个点P(x,y)P(x, y)P(x,y)的方向场斜率是f(x,y)f(x, y)f(x,y), 那么我们可以在这个点画出一条小线段, 称为线素, 这条线段的斜率就是f(x,y)f(x, y)f(x,y). 通过在区域内的多个点上画出这样的线段, 我们就可以得到一个近似的积分曲线的图像. 这种方法被称为方向场法.

当然在构造方向场的时候, 我们也不必逐点地计算斜率. 我们可以固定一些斜率值, 然后求出f(x,y)=kf(x, y)=kf(x,y)=k对应的曲线, 这些曲线被称为等斜率线. 通过画出这些等斜率线, 我们就可以得到一个更清晰的方向场图像, 从而更好地理解微分方程的解的行为.
